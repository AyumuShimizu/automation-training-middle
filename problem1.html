01Middle Class Problem : Random Checkbox

Requirements

  • Randomly check the checkboxes 1 to 10.
    （1から10のチェックボックスをONにしてください）
  • Check at least one check box.
    （チェックボックスは、最低1つ以上はONにしてください）
  • Randomly check the checkboxes and then test for checks.
    （チェックボックスをランダムにチェックした後、チェックの有無をテストしてください。）
  • Test the bool values for all checkboxes.
    （全てのcheckboxのbool値をテストしてください）
