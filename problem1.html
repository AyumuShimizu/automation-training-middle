01Middle Class : Random Checkbox

Problem1

  • Create a method to randomly turn on check boxes 01 to 10.
    （チェックボックス01から10をランダムにONにするメソッドを作成してください）
  • Make sure there is at least one check box on and off for each.
    （チェックボックスはONとOFFそれぞれ1つ以上存在ようにしてください）
  • Randomly check and then automate the test with the Value of all checkboxes.
    （ランダムにチェックした後、全チェックボックスのValueをテストを自動化してください）
  • Automate the test that the bool value labels for all checkboxes are correct.
    （全チェックボックスのbool値のラベルが正しいことをテストを自動化してください）
